## Технические детали:

### Подход к генерации:
1. **Static Site Generation (SSG)** - для основных страниц:
   - Главная, города, виды техники, топ-20 брендов
   - Генерируются один раз при деплое
   
2. **Server-Side Rendering (SSR)** - для вложенных страниц:
   - Все комбинации город+техника, бренд+техника, город+бренд+техника
   - Генерируются по запросу
   
3. **Incremental Static Regeneration (ISR)** - кеширование на Vercel:
   - Динамические страницы кешируются после первого запроса
   - Автоматическое обновление кеша при изменениях

### Фильтрация контента:
- На странице бренда показываются только те виды техники, которые этот бренд производит
- На странице техники показываются только те бренды, которые производят этот вид
- Маппинг определен в `lib/data/serviceBrands.ts`

### SEO оптимизация:
- Каждая страница имеет уникальный title, description, h1
- Динамический контент адаптируется под комбинацию (город/бренд/техника)
- Schema.org разметка (LocalBusiness, Service, BreadcrumbList)
- Sitemap с приоритетами (homepage: 1.0, города+техника: 0.9, и т.д.)
